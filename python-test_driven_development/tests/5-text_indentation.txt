The ``text_indentation`` module
======================

Using ``text_indentation``
-------------------

Firstly, import ``text_indentation`` from the `5-text_indentation.py`` file:

    >>> indent = __import__('5-text_indentation').text_indentation

Tests:

    >>> indent("What? How are you. Really     ")
    What?
    <BLANKLINE>
    How are you.
    <BLANKLINE>
    Really

    >>> indent("   Yes: Thanks:")
    Yes:
    <BLANKLINE>
    Thanks:

    Test - text is not string
