The ``MyList`` module
======================

Using ``print_sorted``
-------------------


First, import ``MyList`` from the ``1-my_list`` file:

    >>> MyList = __import__('1-my_list').MyList

Tests:

>>> my_list = MyList([1, 2, 7, 5, 3])
>>> my_list.print_sorted()
[1, 2, 3, 5, 7]

>>> my_list = MyList([3, 1, 4, 2])
>>> print(my_list)
[3, 1, 4, 2]
>>> my_list.print_sorted()
[1, 2, 3, 4]
